### Installation

[F8Photo.ph UI](https://f8photo.netlify.app) requires [Node.js](https://nodejs.org/) v10+ to run.

Installation
```sh
$ npm install
or
$ yarn
```

Development
```sh
$ npm run dev
or
$ yarn dev
```

Build

```sh
$ npm run build
or
$ yarn build
```

Serve
```sh
$ npm run serve
or
$ yarn serve
```
